Minecraft

Jogador de Minecraft faz o impossível e cria computador 32-bit funcional com 2 kB de RAM dentro do game e impressiona comunidade

Faz a sequência de Fibonacci mas [ainda] não roda Doom

A popularidade de Minecraft fez com que essa licença se tornasse uma das mais bem-sucedidas da história dos videogames. Por esse motivo, seu criador até anunciou recentemente uma sequência espiritual em post na rede social X. Porém, a notícia de hoje não é essa, mas sim a surpreendente construção que um usuário conseguiu criar dentro do mundo do Minecraft: um computador funcional de 32 bits com 2 kB de RAM.

Como relatou o portal PC Gamer, Ryan Boulds é um graduado em ciência da computação que começou o projeto por diversão no ano passado. Notavelmente, seu computador dentro do Minecraft não é apenas funcional, mas também programável, capaz de executar código assembly projetado pelo próprio Boulds. Portanto, ele pode realizar adição, subtração, multiplicação e divisão, mas também pode realizar operações mais complexas que lhe permitem manipular bits.

Ainda não consegue rodar Doom

Um dos primeiros programas que ele executou foi a sequência de Fibonacci. Em resposta à curiosidade dos usuários, o criador destacou que ainda não está apto a rodar Doom, mas confirmou dois aspectos: por um lado, sua intenção de levar o título para sua plataforma dentro do Minecraft; por outro lado, a capacidade do seu computador de processar código de máquina linha por linha. Além disso, ele também revelou que o game roda em circuitos redstone no Minecraft vanilla, mas admitiu que usa mods para otimizar o desempenho.

@ryan_boulds My Minecraft Redstone computer is now in the final stages of construction! As a Computer Science Graduate with a minor in Mathematics, I have been building a 32-bit computer in Minecraft over the past few months. When this is done, it is going to be able to run my own assembly code that will be programmable by the user. It will be able to Add, Subtract, Multiply, Divide, Modulo, Bitshift Left, Bitshift Right, Rotate Right, and Rotate Left.  It will also be able to compare two numbers to see if a number is greater than, less than, or equal to the other number.  It will also eventually be able to do loops! The Arithmetic Logic Unit has already been built which includes a dedicated multiplication unit and a dedicated division/modulo unit that uses a shift and subtract algorithm.  It also has 16 registers that it can read and write to which hold 32 bits each.  Currently, it can run machine code when entered line by line, but I hope one day I will be able to have it run everything pre-programmed automatically. It will also be able to convert Binary Numbers to Binary-Coded-Decimal which can then be converted into decimals (base-10) and displayed on seven-segment-displays. However, for testing, I am using Hexadecimal Seven-Segment-Displays since they are much easier to build, and they take up significantly less space! The computer is about halfway done, and it is already incredibly massive! Boulds acrescentou que a construção do PC não foi totalmente manual, pois ele usou ferramentas como o World Edit para replicar circuitos. Assim, o design final é simétrico e suspenso acima do mundo do jogo, fazendo com que pareça uma nave alienígena. Além disso, o autor do experimento compartilha o progresso e as explicações em sua conta pessoal no TikTok, um perfil que servirá para todos os fãs de matemática, engenharia elétrica e ciência da computação conhecerem os detalhes de um projeto emocionante.
